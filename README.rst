=========================================================
Unix Time Counter
=========================================================

一个实时显示当前Unix时间，目标时间以及距离目标时间的剩余时间的简单程序。精确到秒



Requirements
============

- 易语言(e.exe)

=========================

See the `package homepage`_ for helpful hints relating to downloading
and installing pyswarm.
